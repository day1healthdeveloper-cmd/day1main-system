# Benefit Limits

This folder contains benefit limit definitions and annual caps.

## Documents to Add

- Annual benefit limits by plan
- Per-visit limits
- Lifetime limits
- Family vs individual limits
- Sub-limits (e.g., dental, optical)
- Benefit reset rules (annual, lifetime)

## Format

- PDF documents
- Excel spreadsheets

## Usage

These limits are used in the benefit_usage table and validation logic to prevent over-utilization.
