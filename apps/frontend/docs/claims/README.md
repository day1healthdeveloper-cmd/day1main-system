# Claims Documentation

This folder contains all claims-related company documents including procedures, guidelines, forms, tariffs, and workflows.

## Folder Structure

- **procedures/** - Claims processing procedures and step-by-step guides
- **guidelines/** - Adjudication guidelines and decision-making rules
- **forms/** - Claim forms and templates (PDF/Word)
- **tariffs/** - Tariff codes, pricing schedules, and rate tables
- **rejection-codes/** - Rejection code definitions and usage guidelines
- **workflows/** - Process flowcharts and decision trees
- **pre-authorization/** - Pre-authorization and Guarantee of Payment (GOP) documents

## Usage

These documents are used as the source of truth for implementing claims processing logic in the system. When implementing features, developers should reference these documents to ensure accuracy.

## Document Types

- PDF documents
- Word documents
- Excel spreadsheets
- Images (flowcharts, diagrams)
